# Kawaii Style

Big sparkly anime-inspired eyes with vibrant colors and kawaii charm.

## Features

- **SVG format** — scales infinitely from Tidbyt (64x32) to TRMNL (800x480) and beyond
- **10 emotions** — full set of expressive states
- **Vibrant colors** — blue eyes with colorful accents (red for angry, yellow for excited, green for focused)
- **Kawaii details** — blush, sparkles, and cute mouth styles

## Emotions

- `neutral` — calm, centered gaze
- `happy` — big sparkly eyes with cat smile
- `sad` — droopy eyes with tear-like iris
- `angry` — fiery red eyes with gritted teeth
- `surprised` — wide eyes with open mouth
- `worried` — raised brows with sweat drop
- `sleepy` — closed happy arcs with sleep bubbles
- `excited` — HUGE golden eyes with maximum sparkles
- `confused` — wonky tilted eyes with question mark
- `focused` — sharp green eyes with determination lines

## Installation

```bash
glint style install @sethgho/kawaii
```

## Usage

```bash
glint show happy --style kawaii
```
